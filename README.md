## 🎮 游戏简介

**掼蛋** 是一种流行于中国江苏、安徽等地的四人纸牌游戏，使用两副扑克牌（108张），四人分成两组对抗。本游戏实现了：

- **人工玩家** vs **3个AI对手**（含1个AI队友）
- 完全离线运行，**零网络流量**
- PWA 技术，可安装到手机桌面，像原生 App 一样使用
- 纯前端实现，无需服务器

## ✨ 功能特性

### 游戏机制
- ✅ 完整掼蛋规则：逢人配、顺子、同花、同花顺、炸弹、王炸
- ✅ 逢人配（级牌万能牌）全面支持 — AI 和人类均可使用
- ✅ 升级制：从2打到A，按名次升降级
- ✅ 自动结算+统计

### AI 玩家
- ✅ 阶段感知策略（前期/中期/后期不同打法）
- ✅ 手牌评估（牌多出组合、牌少争头游）
- ✅ 炸弹管理（对手快赢才炸、保留战略资源）
- ✅ 团队意识（不压队友）
- ✅ 使用记牌器推断对手手牌

### 用户体验
- ✅ 双排手牌布局（27张牌全部可见）
- ✅ 4套色彩主题（经典绿🌿 / 深海蓝🌊 / 暗夜紫🌙 / 晨曦红🌅）
- ✅ 玩家昵称和头像自定义
- ✅ 记牌器（实时显示各点数剩余牌数）
- ✅ 游戏统计（自动保存胜率/排名/炸弹数）
- ✅ 音效反馈（Web Audio 合成，无需音频文件）
- ✅ 触感震动（移动端支持）
- ✅ 逢人配呼吸灯动画
- ✅ 完全离线 PWA

### 数据持久化
- ✅ 设置本地保存 (`localStorage`)
- ✅ 战绩统计自动保存（可开关）
- ✅ 每局结束后可手动继续或自动继续（可配置）

## 📱 安装到手机

本游戏是 **PWA（Progressive Web App）**，可以安装到手机桌面：

### Android / 鸿蒙 HarmonyOS
1. 在浏览器中打开游戏页面
2. 点击浏览器菜单（通常是右上角三个点）
3. 选择 **"添加到主屏幕"** 或 **"安装应用"**
4. 桌面将出现掼蛋图标，点击即可启动

### iOS
1. 在 Safari 中打开游戏页面
2. 点击底部分享按钮
3. 选择 **"添加到主屏幕"**
4. 设置名称后添加

安装后**完全离线可用**，无需任何网络连接。

## 📖 游戏规则

### 基本规则
- **人数**：4人，两人一组对抗（你+AI队友 vs 2个AI对手）
- **座位**：你对面的玩家是你的队友
- **牌数**：两副扑克牌共108张，每人27张
- **大小**：3 < 4 < ... < 10 < J < Q < K < A < 2 < 小王 < 大王

### 首出规则
- 第一局持有 **♠2（黑桃2）** 的玩家先出
- 后续局由上一局的 **头游**（第一名）先出

### 逢人配（万能牌）
- 当前级牌的 **所有8张牌**（2副×4花色）都是万能牌
- 万能牌可以替代任何牌组成牌型
- 万能牌有金色边框和呼吸灯效果，容易识别

### 牌型（从小到大排序）
| 牌型 | 说明 | 张数 |
|------|------|------|
| 单张 | 任意单张牌 | 1 |
| 对子 | 两张同点数 | 2 |
| 三同张 | 三张同点数 | 3 |
| 三带一 | 三同张+单张 | 4 |
| 三带对 | 三同张+对子 | 5 |
| 顺子 | 5张连续点数 | 5 |
| 同花 | 5张同花色（非顺子） | 5 |
| 同花顺 | 5张同花色且连续 | 5 |
| 炸弹 | 4-8张同点数 | 4-8 |
| 王炸 | 4张王 | 4 |

### 出牌规则
1. 同牌型才能互相比较，点数大的赢
2. **炸弹可以压任何非炸弹牌型**
3. **同花顺 > 普通顺子**
4. 压不过时可以选择 **过牌**
5. 连续3家过牌，最后出牌者赢得本轮并自由出牌

### 升级规则
| 名次 | 升级 |
|------|------|
| 头游+二游（双上） | 升3级 🎉 |
| 头游+三游 | 升2级 👍 |
| 头游+末游（保头） | 升1级 ✅ |
| 二游+三游（中游） | 平局 🤝 |
| 二游+末游（被双下） | 降1级 ⬇ |
| 三游+末游（双下） | 降2级 ⬇⬇ |

### 胜利条件
从**2**开始打，一路升级到**A**，在A级获胜即通关！

## 🎯 操作说明

### 基本操作
| 操作 | 方法 |
|------|------|
| 选中牌 | 点击手牌（金色高亮 + 上弹） |
| 取消选中 | 再次点击已选中的牌 |
| 清除所有选中 | 点击 **✕ 清除** 按钮 |
| 出牌 | 选好牌后点击 **▶ 出牌** |
| 过牌 | 点击 **⏭ 过牌**（不是首出时可用） |
| 提示 | 点击 **💡 提示** 自动选好最小能出的牌 |

### 顶部按钮
| 按钮 | 功能 |
|------|------|
| ⚙ | 游戏设置（昵称/头像/主题/开关） |
| 📊 | 记牌器 + 战绩统计 |
| 🔊/🔇 | 音效开关 |

### 设置面板
| 设置项 | 说明 |
|--------|------|
| 头像 | 12种 Emoji 可选 |
| 昵称 | 最多8个字符 |
| 主题配色 | 4种颜色方案 |
| 自动保存战绩 | 关掉后成绩不计入统计 |
| 自动继续下一局 | 关掉后每局结束需手动点击 |

## 🛠 技术栈

| 技术 | 用途 |
|------|------|
| **原生 JavaScript** | 零依赖，纯手写 |
| **HTML5 + CSS3** | 界面布局与样式 |
| **PWA / Service Worker** | 离线运行 + 可安装 |
| **Web Audio API** | 音效合成 |
| **Vibration API** | 触感反馈 |
| **CSS Custom Properties** | 动态主题切换 |
| **localStorage** | 设置与统计持久化 |

### 设计原则
- 🔌 **零依赖**：不依赖任何第三方库或框架
- 📶 **零网络**：所有代码在本地运行，无需网络
- 📱 **移动优先**：针对手机触屏优化
- 🎨 **可主题化**：4套配色通过 CSS 变量动态切换

## 🚢 构建部署

本游戏是纯静态页面，可直接部署到任何静态文件服务器：

1. 将整个 `guandan-pwa/` 文件夹上传到服务器
2. 确保服务器支持 `index.html` 作为默认页面
3. 确保 Service Worker 有适当的 `Service-Worker-Allowed` 头部

**推荐部署平台：**
- GitHub Pages
- Netlify
- Vercel
- 任何静态文件服务器（Nginx / Apache 等）
